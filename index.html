Cyphrus

        

          A unified, terminal-driven interface for managing and interacting with locally hosted AI models. Designed for developers who want full control over their workflow, Cyphrus enables seamless switching between models, efficient execution of tasks, and fine-grained performance tuning on your own system.

          Built with a modular architecture, it allows easy integration of new models while maintaining a lightweight and responsive experience. By eliminating reliance on external APIs, Cyphrus prioritizes privacy, reliability, and offline capability—bringing powerful AI workflows entirely under your control.

About

SYSTEMS

    

      I'm a systems-focused software engineer driven by a deep curiosity for how things work beneath the surface. My work spans low-level programming in C/C++, operating systems concepts like memory management, concurrency, and system calls, as well as building performant, real-world applications with modern web technologies. I approach engineering with a strong emphasis on control, efficiency, and clarity—whether it's optimizing execution through profiling and debugging, designing modular architectures, or working with tools like Linux, Git, Docker, and build systems.
      I believe great software comes from understanding both the hardware and the abstraction layers above it. From implementing data structures and multithreaded systems to developing full-stack interfaces and terminal-driven tools like Cyphrus, I aim to build systems that are not just functional, but robust, scalable, and thoughtfully engineered. Every project I take on is an opportunity to push deeper into performance, reliability, and design precision—building systems that last.
